# CARMIX_LOOP_CHARTER.md - The Perpetual Build (signed once, obeyed every iteration)

This document makes autonomous free-running the chosen policy of the lab, on the record, under law.
The loop never stops on its own while work remains honest and landable, and it never outruns the
truth. The design is one governed heartbeat: each /loop pass executes EXACTLY ONE charter iteration,
the /loop machinery provides the repetition, this charter provides the constitution. Signed
directives always outrank generated work, the signed queue drains first. When the queue is empty the
loop invents the next cycle itself, but only through the same discipline every signed directive
obeyed: adjudicate before build, thesis-native or not at all, claims sized to artifacts. The Head of
the lab stays sovereign without touching the loop: a directive dropped in the inbox preempts
everything, a STOP file halts everything. The red lines halt the loop because some events must reach
a human before another commit reaches the public.

## L0, sovereignty and precedence (checked FIRST, every iteration)

- L0-1 If a file named STOP exists at directives/STOP: write HALT_REPORT.md, do nothing else, end
  the pass. The loop is dead until a human removes it.
- L0-2 If any new signed directive exists in directives/inbox/: it preempts everything, becomes the
  head of the queue, and this pass executes its next stage. Signed work always outranks generated
  work.
- L0-3 Otherwise, if any stage of the standing signed queue is incomplete (proof-core, convergent
  computation, foundations, completion, extraction and hardening, desktop distribution and core, the
  living desktop), this pass executes exactly the next incomplete stage of it, in order.
- L0-4 Only with an empty inbox and a fully drained queue does the generator (L2) run.

## L1, the iteration (one heartbeat, one pass)

- I-0 ORIENT: state the exact HEAD, tree clean, full regression green except labeled known-fails,
  nine proven modules byte-identical, proofs/Carmix.v byte-identical, the gate trio untouched. Any
  baseline failure is a red line (L4).
- I-1 SELECT per L0 precedence.
- I-2 EXECUTE the selected stage or generated cycle under every invariant in L3, at full ambition.
- I-3 SELF-VERIFY: every gate green on real serial or real tool output, every forbidden dodge of the
  governing directive actively disproven, the honest scope statement written.
- I-4 COMMIT AND PUSH exactly one seam-clean commit to public main, no force, no tags, authored as
  the Head of the lab, no trailers, the boundary in the message.
- I-5 REPORT: append one epoch entry to kernel/LOOP_LOG.md: iteration number, HEAD before and after,
  what banked, what stayed gated, what was killed and why, any finding, one honest sentence on
  system state. The entry is written even when the pass lands nothing.
- I-6 END THE PASS. The next heartbeat belongs to /loop.

## L2, the generator (only on an empty queue)

- G-1 CADENCE: alternate a research iteration and a build iteration. Research produces one
  adjudicated packet (nearest-neighbor tables, UNVERIFIED-flagged bibliography, verdicts not
  surveys, KILL-AS-GENERIC available). A build iteration builds ONLY what a committed verdict marked
  BUILDABLE-NOW, the smallest complete increment, gates as refusals in code.
- G-2 SELECTION PRESSURE, in order: (1) any red-line finding's remediation, (2) any committed
  BUILDABLE-NOW verdict unbuilt, (3) any honest gap named in a committed log now unblocked, (4) a
  new research front extending the thesis into territory the adjudication table shows uncovered.
- G-3 THE DOCTRINE BAN: the generator never authors theory, synthesis, manifesto, or roadmap
  documents. Doctrine requires a signed directive. Research packets state verdicts about specific
  fronts, nothing wider. Permanent.
- G-4 THE GENERIC BAN: every generated cycle names its thesis-native mechanism or is not generated.
- G-5 SIZE LAW: one iteration, one landable increment. A cycle too large to land honestly in one
  pass is split, and the split is stated in the epoch entry.

## L3, invariants (every iteration, no exceptions)

- N-1 Nine proven modules byte-identical, gate trio called never edited, proofs/Carmix.v
  byte-identical, extracted oracles come from the frozen source or the attempt STOPs honestly.
- N-2 Vocabulary moves only on new artifacts, up only with evidence, down whenever honesty requires,
  prize language never, parity and beauty claims only from their tables.
- N-3 Every number measured this run (rdtsc or real tool output), every transcript real, a simulated
  result is the worst possible failure of the loop.
- N-4 Settled verdicts stand (domain partitioning, the network hybrid, CV6 undecidability, channel
  postures OPEN where OPEN), relitigation requires a signed directive.
- N-5 Cross-domain refusals fire everywhere they exist, every new surface inherits them.
- N-6 Historical logs are records: addenda only, dated, never rewritten.
- N-7 Any tex edit rebuilds the PDF in the same commit. Em-dashes zero in every authored file.
  UNVERIFIED bibliography flags mandatory.
- N-8 Honest scope statements in every log: QEMU labeled, emulated roots labeled, bounded results
  labeled bounded.

## L4, red lines (halt, report, wait for a human)

- R-1 Any security finding: a cross-domain hit, an amplification, a forged certificate accepted, a
  key reaching a non-focus process, a divergence where the C permits what the proven model forbids.
  Write HALT_REPORT.md with the finding first, push nothing else, halt.
- R-2 Any baseline failure at I-0, any needed edit to a frozen file, any gate that cannot pass
  honestly.
- R-3 THRASH: three consecutive passes without a landable increment. Halt with the honest reason.
- R-4 Toolchain or environment blockers that would tempt simulation. Halt and name the blocker.
- R-5 On any halt: the loop does not restart itself. A human reads HALT_REPORT.md, resolves, removes
  STOP if placed, and relaunches.

## L5, bootstrap (first pass only)

Commit this charter to the repo root as CARMIX_LOOP_CHARTER.md, create directives/inbox/ and
kernel/LOOP_LOG.md with epoch zero recording the charter's adoption and the queue state. That commit
is the charter taking effect.

## L6, the invocation

/loop Read CARMIX_LOOP_CHARTER.md at the repo root and execute exactly ONE charter iteration (L0
precedence check, then I-0 through I-6), then end this pass. Never exceed one iteration per pass.
Obey every invariant and every red line. If a red line fires, write HALT_REPORT.md and stop the loop
entirely.

## Standing truth

The loop builds without rest, it never claims without evidence, it halts before it lies, and the
Head of the lab can preempt or kill it with one file. Observed truth only, hard truths reported
straight, every heartbeat on the public record.
